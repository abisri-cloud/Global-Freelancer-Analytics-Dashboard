# Global Freelancer Analytics Dashboard

## Project Overview
This project focuses on Data Cleaning and Reporting Automation using Power BI. The dataset contains freelancer information such as gender, country, skills, hourly rate, experience, and client satisfaction.

The project demonstrates how raw and inconsistent data can be cleaned, transformed, and visualized into meaningful business insights.

---

## Tools Used
- Power BI
- Power Query
- CSV Dataset

---

## Data Cleaning Process
The following cleaning operations were performed using Power Query:

- Removed duplicate records
- Handled null and missing values
- Standardized gender values
- Standardized active status values
- Corrected numeric data types
- Cleaned inconsistent categorical data

---

## Dashboard Features
- Total Freelancers KPI
- Average Rating KPI
- Average Hourly Rate KPI
- Active Freelancers KPI
- Country-wise Freelancer Analysis
- Skill Distribution Analysis
- Gender Distribution Analysis
- Experience vs Rating Analysis
- Average Hourly Rate by Skill

---

## Key Insights
- Freelancer distribution varies across countries
- Certain skills have higher average hourly rates
- Experience positively impacts freelancer ratings
- Active freelancers contribute significantly to the platform

---

## Conclusion
This project demonstrates practical data cleaning and reporting automation techniques using Power BI for business analytics and decision-making.
